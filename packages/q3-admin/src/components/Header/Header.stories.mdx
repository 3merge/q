# Header

The <code>Header</code> component prints the landmark title
for the page, along with any associated global actions..

<Preview>
  <Story name="With toolbar">
    <Header title="Header with toolbar">
      Toolbar region
    </Header>
  </Story>
</Preview>

It optionally renders a back button component for nested
pages. In such cases, the title size shrinks to further
indicate the nested hierarchy

  <Story name="With back button">
    <Header
      title="Header with back button"
      backComponent={
        <IconButton>
          <KeyboardBackspace />
        </IconButton>
      }
    />
  </Story>
</Preview>
