# useSearch

Returns search results in standerizedld format, serch term
previously used, sorting function and error object.

## Return value

| Property    | Description          |
| ----------- | -------------------- |
| error       | error object or null |
| lastSearch  | string value         |
| Collections | description below    |

The collections is a class holding search result and sort
functions. You can get the result by calling `extract`
method on the Collections.

## Methods

| Method           | Argument                      | Description                 |
| ---------------- | ----------------------------- | --------------------------- |
| `alphabetSort`   | N/A                           | Sort alphabetically         |
| `locationSort`   | { pathnaem: 'yourLocation'}\* | Sort based off the location |
| `collectionSort` | [string]\*                    | Sort by collection names    |

## Arguments

| Type        | Description |
| ----------- | ----------- |
| `string*`   | Search term |
| `[string]*` | Endpoints   |

## Example

```javascript
import React from 'react'
import { useLocation } from '@reach/router'
import { useSearch } from 'q3-hooked'
import Result from './Result'

export default () => {
  const [input, setInput] = React.useState()
  const location = useLocation()
  const { collections, error } = useSearch(input, ['/users', '/blogs'])

  const handleOnChange = (e) => setInput(e.target.value)

  return (
    <div>
      <label>
        <input
          type="text"
          value={input}
          onChange={handleOnChange}
        />
      </label>
      {!error && <Result result={Collections.alphabetSort().locationSort(location).extract())} /> }
    <div>
  )
}
```
